# Tans-pipeline

Simplified transcriptome analysis process

---

### 1. 解压缩

```bash
unzip unzip Tans-pipeline-main.zip
```

### 2.执行权限

```bash
cd Tans-pipeline-main && chmod 777 tans
```

### 3. 配置安装

```bash
./tans -n 
```

### 4.使用以及查看帮助(使用前需要先激活环境)

```bash
conda activate tans
tans --help
```

## **⚠️注意：使用前请先安装conda**
